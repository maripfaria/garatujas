## Conceitos básicos

**Arrays**(Arranjo): É uma estrutura de dados, que serve para agrupar diferentes informações em uma única estrutura. Ele reserva um espaço sequencial na memória para armazenar esses elementos. Cada item da lista ocupa uma posição específica, o que permite que o computador localize e manipule os dados de forma rápida. Ao inves de criar uma variável para cada item, você coloca tudo em um grupo só, usando colchetes [].

Exemplo de declaração:

const listaDeFrutas = ['Maçãs','Uvas','Bananas','Abacaxi','Morangos']; 

**Manipulação de Strings:** É o ato de transformar ou extrair informações de um texto.

Exemplos comuns: Deixar tudo em maiúsculo (.toUpperCase()), contar quantos caracteres tem (.length) ou juntar dois textos (concatenação).

**Funções:** Blocos de código reutilizáveis projetados para realizar tarefas específicas, permitindo estruturar, organizar e evitar a repetição de código
